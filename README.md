E-Learning Frontend (FE)
1. Overview

This project is the Frontend (FE) of an E-Learning platform, built using React, Ant Design, and Tailwind CSS.

The goal of this FE application is to provide a clean, responsive, and user-friendly interface for learners and administrators to interact with the e-learning system.

2. Tech Stack

React – Main frontend framework (component-based architecture)

Ant Design (AntD) – UI components (Table, Form, Modal, Button, etc.)

Tailwind CSS – Utility-first CSS for layout, spacing, and custom styling

React Router – Client-side routing


3. Main Features

View course list

View course details

Watch lessons / learning content


4. UI & Styling Approach

Ant Design is used for:

Tables

Forms & validation

Modals & notifications

Pagination and filters

Tailwind CSS is used for:

Layout (flex, grid)

Spacing, sizing, colors

Custom UI adjustments

Responsive design

=> AntD handles logic-heavy UI, while Tailwind helps with visual customization.
